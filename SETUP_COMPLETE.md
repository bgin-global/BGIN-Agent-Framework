#### 2. Start Development
```bash
# Navigate to project
cd C:\Users\mitch\BGINAI\bgin-ai-mvp

# Install dependencies
npm install
cd frontend && npm install && cd ..
cd backend && npm install && cd ..

# Start services
docker-compose up -d

```
